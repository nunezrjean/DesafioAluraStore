# 📊 AluraStore Latam - Data Analysis

Este proyecto analiza el desempeño de **4 tiendas ficticias de AluraStore Latam** usando **Python, Pandas y Matplotlib**. El objetivo es identificar métricas clave y recomendar cuál tienda debería ser vendida.

---

## 🎯 Objetivos
- 📈 Comparar ingresos totales por tienda  
- ⭐ Evaluar satisfacción del cliente (calificaciones)  
- 🚚 Analizar costos de envío promedio  
- 🛒 Identificar categorías y productos más/menos vendidos  
- 📊 Generar visualizaciones para apoyar decisiones  

---

## 📂 Dataset
Los datos provienen de archivos CSV con información de ventas:

| Archivo       | Contenido principal |
|---------------|---------------------|
| tienda_1.csv  | Ventas Tienda 1     |
| tienda_2.csv  | Ventas Tienda 2     |
| tienda_3.csv  | Ventas Tienda 3     |
| tienda_4.csv  | Ventas Tienda 4     |

**Columnas:** Producto, Categoría, Precio, Costo de envío, Fecha, Vendedor, Lugar, Calificación, Método de pago, Cuotas, Lat/Lon.

## ✅ Conclusión
